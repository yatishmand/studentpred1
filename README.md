# 🎓 Student Academic Analytics Platform

An enterprise-grade predictive analytics and machine learning platform designed to track, monitor, and optimize student performance trajectories. This system integrates advanced behavioral metrics, psychographic risk patterns, and institutional benchmarks using an ensemble Random Forest model and automated AI strategy nodes.

---

## 🚀 Core Platform Architecture & Features

The platform processes student data across multiple dimensions, dynamically computing **26 operational telemetry metrics** across three specialized dashboard categories:

### 📊 Section 1: Academic Forecasting & Performance Predictive Core
* **Predicted Semester GPA:** Dynamic estimation of final results on a 4.0 to 10.0 scale.
* **Backlog Probability Rating:** Machine learning risk index predicting the likelihood of academic drops.
* **Exam Readiness Score:** An aggregate readiness index (0–100%) mapped against daily study consistency.
* **Academic Performance Score:** Overall historical tracking score evaluated from assignment metrics and attendance.
* **Expected Syllabus Coverage:** Predictive time-series mapping of topic completion thresholds.
* **Sessional Marks Evaluation:** Machine learning projection of internal assessment metrics (Out of 20).
* **Curricular Credit Integrity:** Point-scale weight optimization based on target subject credit distribution.
* **Peer Interaction Matrix:** Behavioral classification of collaborative forum interaction data.

### 🧠 Section 2: Cognitive Fatigue & Habit Continuity Indicators
* **Cognitive Fatigue Index:** Risk analyzer calculating cognitive burnout thresholds from sleep-to-study ratios.
* **Focus Attention Span:** Predictive time-span mapping identifying maximum continuous focus blocks.
* **Procrastination Probability:** Behavioral latency matrix calculating assignment delays.
* **Burnout Risk Velocity:** Velocity vector mapping acute stress constraints.
* **Study Efficiency Ratio:** Quantitative evaluation of hours input vs LMS portal click efficiency.
* **Weekend Load Risk Rating:** Workload variance index tracking preparation imbalances.
* **Routine Balance Score:** Homeostasis rating checking sleep, study, and extracurricular parity.
* **Memory Recency Factor:** Cognitive decay metrics predicting active concept retention spans (in days).

### 💼 Section 3: Professional Gateways & Future Career Placements
* **Scholarship Gate Chance:** Eligibility verification matrix checking compliance with institutional criteria.
* **Campus Placement Eligibility:** Automated rule-engine checking cutoff boundaries for corporate shortlists.
* **Career Domain Orientation:** Classification output separating Core Technical capabilities from Management profiles.
* **Target Goal Likelihood:** Prescriptive variance analyzer calculating the probability of hitting target CGPA goals.

## 🛠️ Technological Infrastructure

The backend pipelines are engineered using a streamlined, local serverless architecture ensuring zero-latency operations and high scalability:

* **Machine Learning Architecture:** Scikit-Learn `RandomForestClassifier` ensemble model trained over comprehensive student engagement repositories.
* **Explainable AI (XAI):** Gini Impurity feature importance analytics plotting real-time strategic data drivers.
* **Natural Language Processing Core:** Connected via zero-key execution mapping to a serverless `Meta-Llama-3-8B-Instruct` inference pipeline for customized remediation loops.
* **UI Interface Layer:** Streamlit runtime execution environment styled with custom high-gloss dark slate styling sheets.

## 📂 System Dataset Mapping

The core predictive models parse structural variables across the following key telemetry pipelines:
1.  **LMS Footprints:** `raisedhands`, `VisITedResources`, `AnnouncementsView`, `Discussion`.
2.  **Behavioral Logs:** `StudyHours`, `SleepTime`, `Attendance`, `StageID`, `Gender`.
3.  **Performance Inputs:** `Assignment_Score`, `Study_Consistency`, `Target_CGPA`.
