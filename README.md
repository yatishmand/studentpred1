# 🎓 Student Academic Analytics Platform

> ### 🚀 INSTITUTIONAL METRICS CORE: ACTIVE PIPELINE
> An enterprise-grade predictive analytics and machine learning platform designed to track, monitor, and optimize student performance trajectories. This system integrates advanced behavioral metrics, psychographic risk patterns, and institutional benchmarks using an ensemble Random Forest model and automated AI strategy nodes.

---

## 🛠️ Core Platform Architecture & Features

The platform processes student data across multiple dimensions, dynamically computing **26 operational telemetry metrics** across three specialized dashboard categories:

### 📊 Section 1: Academic Forecasting & Performance Predictive Core
* `🔮 Predicted Semester GPA` — Dynamic estimation of final results on a 4.0 to 10.0 scale based on real-time control metrics.
* `🚨 Backlog Probability Rating` — Machine learning risk index predicting the structural likelihood of academic drops.
* `🛡️ Exam Readiness Score` — An aggregate readiness index (0–100%) mapped against study consistency and assignment variables.
* `🏆 Academic Performance Score` — Overall tracking score evaluated dynamically from assignment metrics, historical inputs, and classroom engagement.
* `📖 Expected Syllabus Coverage` — Predictive time-series mapping of topic completion thresholds based on active parameters.
* `🏅 Sessional Marks Evaluation` — Machine learning projection of internal assessment metrics (Out of 20 marks framework).
* `⚖️ Curricular Credit Integrity` — Point-scale weight optimization mapped against target subject credit distribution.
* `👥 Peer Interaction Matrix` — Behavioral classification of collaborative forum interaction data and engagement velocity.

### 🧠 Section 2: Cognitive Fatigue & Habit Continuity Indicators
* `💤 Cognitive Fatigue Index` — Risk analyzer calculating cognitive burnout thresholds from sleep-to-study ratios.
* `⏱️ Focus Attention Span` — Predictive time-span mapping identifying maximum continuous focus blocks in minutes.
* `⏳ Procrastination Probability` — Behavioral latency matrix calculating assignment submission delays and completion gaps.
* `📉 Burnout Risk Velocity` — Velocity vector mapping acute student stress constraints against workload metrics.
* `📊 Study Efficiency Ratio` — Quantitative evaluation of hours input vs LMS portal click efficiency.
* `🧱 Weekend Load Risk Rating` — Workload variance index tracking preparation imbalances and last-minute load distributions.
* `⚖️ Routine Balance Score` — Homeostasis rating checking sleep, study, and extracurricular parity allocation.
* `🔄 Memory Recency Factor` — Cognitive decay metrics predicting active concept retention spans (in days).

### 💼 Section 3: Professional Gateways & Future Career Placements
* `🎓 Scholarship Gate Chance` — Eligibility verification matrix checking compliance with institutional scholarship criteria.
* `🚀 Campus Placement Eligibility` — Automated rule-engine checking cutoff boundaries for enterprise and corporate shortlists.
* `🎯 Career Domain Orientation` — Classification output separating Core Technical engineering capabilities from Management profiles.
* `🎯 Target Goal Likelihood` — Prescriptive variance analyzer calculating the exact probability of hitting target CGPA goals.

## 📂 System Dataset Mapping

The core predictive models parse structural variables across the following key telemetry pipelines:
1. **LMS Footprints:** Classroom Interaction (`raisedhands`), Database Portal Clicks (`VisITedResources`), Notification Logs (`AnnouncementsView`), Group Discussion (`Discussion`).
2. **Behavioral Logs:** Continuous Study Hours (`StudyHours`), Rest Cycle (`SleepTime`), Campus Record (`Attendance`), Domain Level (`StageID`).
3. **Performance Inputs:** Task Submissions (`Assignment_Score`), Continuity (`Study_Consistency`), Target Parameter (`Target_CGPA`).
